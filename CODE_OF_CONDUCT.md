# Code of Conduct

This project aims to be a practical, welcoming place for contributors and users. Everyone participating in issues, pull requests, discussions, and other project spaces is expected to help keep the community respectful and professional.

## Expected Behavior

- Be considerate, patient, and constructive.
- Assume good intent, especially when reviewing code or discussing bugs.
- Give feedback on ideas and changes, not on people.
- Respect different levels of experience and different communication styles.
- Keep discussions relevant to the project and its users.

## Unacceptable Behavior

The following behavior is not acceptable:

- Harassment, intimidation, threats, or personal attacks.
- Discriminatory language or conduct.
- Sexualized language or imagery.
- Publishing private information without explicit permission.
- Repeated disruption of project discussions after being asked to stop.

## Enforcement

Maintainers may moderate, edit, hide, or remove comments, issues, pull requests, or other contributions that violate this Code of Conduct. They may also temporarily or permanently restrict participation in the project.

If you need to report a concern, contact the project maintainers through a private channel listed in the repository, if available. If no private channel is listed, open a GitHub issue with only the information you are comfortable sharing publicly and ask for maintainer follow-up.

## Scope

This Code of Conduct applies in project spaces and in public spaces when someone is representing the project or community.
